    This is a processed version of a DVD that was made from a VHS tape. The quality is good enough for a VHS from 1987.

    My edits using Avisynth, ffmpeg and MKVToolnix: deinterlaced with QTGMC for a smooth 59.94fps, deblocked, resized to 4:3 using Spline36Resize, cropped the edges with ugly video artifacts, saved to MP4 AVC (CRF 18), added song titles to existing chapters, added text info file. The audio is identical with that DVD release.
